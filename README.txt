Como fazer a build: 

Executando o .exe:
Depois de verificar que os arquivos 'wordfreq.exe', 'franksenstein_chpt4' e 'stop_words' estão no mesmo diretório:
No terminal:
	- Navegue até o diretório dos arquivos usando cd
	- Execute o arquivo '.exe' com ./wordfreq.exe
	
Configurando e compilando:
-Baixe e instale o EiffelStudio: site oficial: 
-Abra o EiffelStudio, selecione 'file'>'open Project'>encontre o arquivo com final .ecf
-Compile o projeto clicando em 'Project'>'Compile W C'
