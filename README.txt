Como fazer a build: 

Executando o .exe:
Depois de verificar que os arquivos 'wordfreq.exe', 'frankstein_chpt4.txt' e 'stop_words.txt' estão no mesmo diretório:
No terminal:
	- Navegue até o diretório dos arquivos usando cd
	- Execute o arquivo '.exe' com wordfreq.exe ou ./wordfreq.exe
	- filepath: o default é 'frankstein_chpt4.txt', dá pra usar outros textos, caso ele não encontre, usará o default
	
Configurando e compilando:
-Baixe e instale o EiffelStudio: site oficial: https://www.eiffel.com/eiffelstudio/
-Abra o EiffelStudio, selecione 'file'>'open Project'>encontre o arquivo com final .ecf
-Compile o projeto clicando em 'Project'>'Compile W C'
